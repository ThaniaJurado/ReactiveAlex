# Reactive Alex
This app is intended to be a re-make of Alex1.2, but now in React Native instead of Android Studio

# Features
- User configuration (internal storage)
- Email use
- SMS messages
- Geolocation

NOTE: This app is still under construction

## Coming soon (pending improvements)
